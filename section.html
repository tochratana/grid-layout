Master layout with 5 essential concepts

1. Grid Container

Use display: grid on a parent element to create a grid context. It enables the use of grid properties on child elements.

2. Grid Template Columns/Rows

Define columns and rows using grid-template-columns and grid-template-rows. Example: grid-template-columns: repeat(3, 1fr);

3. Grid Gap

Use gap to add space between rows and columns. You can set separate values for row and column gaps using row-gap and column-gap.

4. Grid Item Placement

Use grid-column and grid-row on children to control their placement. You can also span across multiple columns or rows.

5. Implicit vs Explicit Grid

An explicit grid is defined using template properties. If items exceed the defined grid, implicit tracks are created automatically to hold them.
